Prework Study Guide

HTML

  • HTML gives the content structure.
  • !DOCTYPE is not case sensitive but HTML elements are.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • An element refers to the entire element from opening to closing tag.
  • A tag refers only to what appears inside the angle brackets.
  • The h1 to h6 elements represent the level of heading a given text block represents.
  • The p element represents a paragraph or block of text.
  • HTML uses the id attribute as a way to identify elements and select them.

CSS

  • CSS stands for Cascading Style Sheets.
  • A programming language used by devs to define how webpage content is presented to users.
  • CSS defines the colors, aesthetics, visual layout, and organization.
  • Inline CSS applies styles to a specific HTML tag by using a style attribute with a CSS rule; less flexible.
  • External CSS style sheets are more common because it allows decs to keep all their CSS rules in a separate file, which makes design changes easier.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Multiple selectors can be combined into a single CSS rule with a comma, helping us write Do Not Repeat Yourself (DRY) code.
  • DRY can greatly reduce the overal size and complexity of a codebase.
  • The period preceding a CSS rule designates this as a class selector; any HTML element assigned to the class will gain these property assignments from CSS.
